# HNU CS Tutorial 2026 - 3

本项目为 2026 年计算机系统或相关课程实验项目的代码、测试数据及分析报告仓库。仓库包含了有关缓存优化、矩阵乘法性能测试相关的不同实验：

## 目录结构

- **2.1_a/**: 矩阵乘法不同循环顺序的基础代码 (`gemm_loop_order.c`) 及相关验证。
- **2.1_bc/**: 矩阵乘法循环顺序之争与 Cache Miss 影响实验。包含 `cachegrind` 和 `perf` 的测试脚本、实验日志与分析报告。
- **2.1_de/**: 进一步分析分块矩阵乘法（循环顺序优化）、组索引机制对性能的影响及深入的探索，包含生成 PPT 和扩展探讨的代码、脚本文件。
- **3.1_ab/**: 存储层次金字塔性能实测，使用指针追逐等方法验证不同缓存层级的实际延迟和带宽表现。
- **3.1_c/**: 工作集大小扫描与虚拟机环境性能差异实验。
- **3.1_de/**: 存储设备（SSD 与 HDD）的随机 4K 读写性能对比与报告 (`report_de.md`)。
- **presentations/**: 本项目期间产出的各阶段和最终版的幻灯片汇报演示文稿 (`.pptx`)。
- **docs/**: 其他文档及设计类资料（如 `octo-code-DESIGN.md`）。

## 实验概览

1. **实验 2.1**:
    探讨矩阵乘法在不同的循环展开结构及其对于缓存失效率 (Cache Miss Rate) 和实际执行时间的影响，验证多级存储结构下的程序优化原理。
2. **实验 3.1**:
    结合具体的系统配置，通过实验测量各级缓存及主存的延迟（Latency）与带宽（Bandwidth），对比虚拟机与宿主机环境下的表现差异。

## 运行环境参考

大多数 C 代码由 `gcc` 编译执行，需要 Linux 操作系统提供 `perf` 与 `cachegrind` (Valgrind 工具集) 支持。关于具体环境和硬件指引，请见各目录下的说明文件。
